# 会话摘要模板

在手动编写摘要或审阅 `scripts/context_compressor.py summarize` 的输出时，请使用下面这个结构。

```markdown
# 会话摘要 - YYYY-MM-DD

## 当前状态
- 项目：{项目名称}
- 阶段：{当前开发阶段}
- 阻塞项：{阻塞问题或无}
- 标题：{简短会话标题}

## 已完成
- {已完成任务}

## 关键决定
- {决定及原因}

## 变更文件
- {路径}（变更类型或用途）

## 下一步
- {下一步动作}

## 重要提醒
- {需要保留的配置、环境、命令或约束}

## 原始记录
{可选原始记录，仅在用户明确要求保留时填写}
```

每条要点都尽量具体，这样未来的代理不用重新读完整段对话也能直接接手。
